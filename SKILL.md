# Pain Point Hunter 🔍 — 竞品痛点分析工具

输入一款游戏的名字，自动抓取真实用户评价，输出 TOP10 痛点清单 + 评分分布 + 策划洞察。

---

## 数据来源

- **App Store**（iTunes Reviews API，RSS Feed）
- **TapTap**（社区评价 API，含中文玩家真实反馈）

---

## 使用方式

### 触发方式

直接输入游戏名：

```
分析一下《原神》的玩家痛点
《王者荣耀》差评集中在哪些问题
卧龙苍天陨落玩家都在吐槽什么
帮我分析一下明日方舟的差评
```

### 执行流程

**Step 1：抓取评价**
- App Store 评分 + 评论（最新50条）
- TapTap 评分 + 评论（热门50条）

**Step 2：AI 分析**
- 聚类同类抱怨
- 统计高频关键词
- 识别情感极性（好评/差评分布）

**Step 3：输出结构化报告**

## 关键词维度（自动识别）

| 维度 | 关键词示例 |
|------|---------|
| 付费/逼氪 | 648、抽奖、保底、逼氪、骗氪 |
| 肝度/体力 | 体力、无内容、长草、重复劳动 |
| 数值/平衡 | 数值膨胀、削弱、职业不平衡 |
| 服务器/技术 | 卡顿、掉线、闪退、发热 |
| 玩法/内容 | 重复、日常一条龙、无创新 |
| 剧情/角色 | 剧情跳过的、角色塑造差 |
| 社交/公会 | 公会没人、匹配慢、鬼服 |

---

## 注意事项

- 抓取需要网络访问，沙盒环境如无法访问App Store/TapTap会自动跳过可用平台
- TapTap评价来自中文玩家社区，对国内游戏分析价值更高
- 数据量越大分析越准，建议同时抓取两个平台
- 抓取结果自动缓存30分钟，避免重复请求
